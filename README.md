# 📊 Data Storage System (Flask + MySQL)

## 📌 Project Description

This is a simple **Flask web application** that allows users to store and view data using a MySQL database.  
It demonstrates basic **CRUD operations (Create & Read)** using Python Flask and MySQL.

The system has two main features:
- 📥 Store Data (Insert user information into database)
- 📄 View Data (Display all stored records in a table)

---

## 🚀 Features

- Simple and clean UI
- Store user details in MySQL database
- View all stored data in table format
- Flask backend with Jinja2 templates
- Basic form handling using POST method
